# Full Stack Fitness App

##  Features
- User authentication (JWT)
- Admin dashboard
- Workout & meal planning

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

##  Project Structure
/frontend
/backend
/admin

## Setup

### Backend
cd backend
npm install
npm start

### Frontend
cd frontend
npm install
npm start

### Admin
cd admin
npm install
npm start
